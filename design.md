# 灵魂星轨 SoulOrbit - 设计文档

## 品牌风格
- **主色调**: 深邃夜空蓝 `#0F172A`（背景）
- **强调色**: 暗金色 `#D4AF37`（按钮/高亮）
- **文字色**: 灰白色 `#E2E8F0`
- **卡片背景**: `#1E293B`（深蓝灰）
- **次级文字**: `#94A3B8`
- **整体风格**: 暗黑神秘学风 (Dark Theme)，星空、塔罗牌、神秘符文

## 屏幕列表

1. **启动引导页 (Splash)** - 纯色背景+Logo+"触碰星轨"按钮
2. **灵魂测试页 (Test)** - 30题依次渲染，进度条，A/B选项卡
3. **能量主页 (Home)** - 每日塔罗日签，用户人格卡片，底部导航
4. **羁绊沙盘页 (Sandbox)** - 好友合盘列表，添加好友
5. **合盘结果页 (Relation Result)** - 双人羁绊卡牌与神明箴言
6. **付费墙弹窗 (Paywall)** - 展示两个付费选项
7. **深度报告页 (Deep Report)** - 4章完整深度解析报告

## 主要用户流程

1. 首次启动 → 启动引导页 → 灵魂测试页（30题）→ 计算MBTI → 能量主页
2. 能量主页 → 查看每日日签 → 点击"深度报告"→ 付费墙弹窗（展示）→ 深度报告页
3. 底部导航 → 羁绊沙盘 → 添加好友 → 查看合盘结果

## 字体风格
- 标题：粗体，大字号，金色或白色
- 正文：中等字号，灰白色
- 箴言/引用：斜体，金色，带有神秘感

## 关键UI组件
- 星空粒子背景（使用渐变模拟）
- 塔罗牌卡片（带金色边框）
- 神秘符文装饰线条
- 暗金色渐变按钮
- 进度条（测试页）
